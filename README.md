# Family BP Tracker

Family BP Tracker is a web application that allows users to track blood pressure readings for themselves and their family members.

## Planned Features

- Add blood pressure readings manually
- Upload BP monitor photos
- Extract readings from photos
- Family member profiles
- Reading history and trends
- Dashboard with graphs
- Alerts for abnormal readings
- Daily reminders

## Tech Stack (Planned)

- Frontend: React / Next.js
- Backend: FastAPI
- Database: PostgreSQL
- OCR: To be decided
